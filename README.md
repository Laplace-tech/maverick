# Maverick

Maverick is a long-term study repository for building a serious foundation in machine learning, deep learning, and modern AI systems.

## Period

2026-06-11 ~ 2026-09-07

## Goal

Build the foundation required for ML Engineer / Research Engineer growth.

## Tracks

- MML: Mathematics for Machine Learning
- ISLP: Classical statistical learning and traditional machine learning
- D2L: Modern deep learning through implementation and experiment logs
- Hypermodern: LLM, Agent, RAG, Multimodal, Serving, Evaluation, and MLOps

## Rule

Read.  
Derive.  
Implement.  
Experiment.  
Commit.  
Review.

## Environment

This repository does not depend on the `d2l` Python package.

D2L is used as a web textbook, and implementations are written directly with PyTorch.
